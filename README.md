## biblenote

聖書に書かれていることを、自分なりに整理できるアプリです。<br>
聖書の内容を引用し、引用句についてノートをまとめることができます。<br>
ノートには写真とタグを登録でき、タグ別にノートの内容を整理できます。<br>
会員同士で友達登録をすると、ノートの内容をお互いに見あい、コメント欄で内容について話すことができます。<br>
